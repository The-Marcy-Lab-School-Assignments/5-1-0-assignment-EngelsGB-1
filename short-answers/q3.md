# Question

In a code snippet, demonstrate a `User` class with at least one:

- class property
- class method
- instance property
- instance method

# Answer

```js
class User {
  static users = []; // Class property

  constructor(name) {
    this.username = name; // Instance property
    User.users.push(this);
  }

  getName() {
    // Instance method
    return this.username;
  }

  static getUsers() {
    // Class method
    return [...User.users];
  }
}
```
